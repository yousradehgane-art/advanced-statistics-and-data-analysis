# Data Wrangling: Healthcare Coverage and Spending Analysis

This folder contains the first homework from the Advanced Statistics and Data Analysis course. The analysis explores healthcare coverage and healthcare spending across U.S. states.

## Research Questions

1. Is there a relationship between healthcare coverage and healthcare spending in the United States?
2. How does the spending distribution change across geographic regions in the United States?
3. Does the relationship between healthcare coverage and healthcare spending change from 2013 to 2014?

## Files

- `healthcare-coverage-spending-analysis.Rmd`: R Markdown file containing the full analysis
- `healthcare-coverage.csv`: healthcare coverage dataset
- `healthcare-spending.csv`: healthcare spending dataset
## Methods Used

- Data cleaning
- Data wrangling
- Wide-to-long data transformation
- Dataset joining
- Coverage proportion calculation
- Healthcare spending per capita calculation
- Scatterplots and boxplots using ggplot2

## Tools Used

- R
- tidyverse
- ggplot2
- ggrepel
- readr
- dplyr
- tidyr
  
## Summary

The analysis investigates how healthcare coverage types relate to healthcare spending per capita across U.S. states. 
It also compares spending distributions across geographic regions and checks whether the coverage-spending
relationship changed between 2013 and 2014.
