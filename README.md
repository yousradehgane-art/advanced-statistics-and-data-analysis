# Advanced Statistics and Data Analysis

This repository contains coursework and applied exercises completed as part of an Advanced Statistics and Data Analysis class. The work covers key statistical learning and data analysis methods, including data wrangling, linear models, generalized linear models, principal component analysis, clustering, and LASSO regularization.

## Topics Covered

- Data wrangling and data cleaning
- Linear models
- Generalized linear models
- Principal Component Analysis
- Clustering methods
- LASSO regularization
- Statistical interpretation and visualization
